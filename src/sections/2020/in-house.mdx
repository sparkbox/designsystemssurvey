## Design Systems Have Similar Elements

Respondents were asked to select all that apply from 19 items and could write in other answers. Over 50% of in-house respondents reported their design system included the following elementsâ€”with color systems appearing in 95% of the design systems.

### What does your design system contain?

1. Color system (94.62%)
1. Typography system (84.62%)
1. Form components (78.46%)
1. Navigation components (76.15%)
1. Spacing system (71.54%)
1. Usage guidelines (70.77%)
1. Grid system (64.62%)
1. Framework-specific components (React, Angular, etc) (60.77%)
1. Accessibility guidelines (60.00%)
1. Design files (59.23%)
1. CSS code (59.23%)
1. Brand guidelines (57.69%)
1. HTML code (53.08%)
1. JavaScript code (53.08%)
1. Layout system (52.31%)

## Design Systems Are More Than Just a Reference

The majority of in-house respondents answered that they have integrated their design system into the codebase in some fashion rather than using it as a standalone reference tool.
